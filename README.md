alice.lips
==========

非公開レポジトリ

公開できるようになった時点で、ディレクトリごと
https://github.com/alice-lips
にコピーする．

## ファイルの管理

`(公開する場)/(プロジェクト名)/`
というディレクトリを作ってその下にプロジェクトのファイルを置くこと．

できれば、
簡単に趣旨を説明するテキストファイル
`(公開する場)/(プロジェクト名)/README.md`
あるいは `README.txt` を置くこと．

(e.g.
次の冬コミは `c87` なので
`c87/project-dummy/`
とする．)

共通して使うかもしれない、または使う予定はまだないものは、トップにある
`misc/`
に置く．
